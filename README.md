# ChatWik
Welcome to ChatWik! This repository contains multiple versions of the ChatWik application, a social chat media platform designed to enhance real-time communication and user engagement. Each version represents a different stage of development, with new features and improvements added progressively.

## Repository Structure
- chatwik-v1/: The first version of ChatWik, implementing core features like group chat, message visibility, user management, and more.
- chatwik-v2/: The second version of ChatWik, planned to include private messaging, additional security features, and performance optimizations.
- More versions will be added as the project evolves.
# Version 1: Overview
In Week Two of development, the core functionality of ChatWik was implemented. This includes:

- Group Chat: Users can view and message everyone on the app. Messages are visible to all users.
- Message Management: Users can update or delete their messages.
- User Directory: View a list of all members and see active users.
- Note: Private chat is not available in version one and is planned for version two.

## Getting Started
### Prerequisites
- Python 3.x
- Flask
- SQLAlchemy
- Other dependencies listed in requirements.txt
## Installation
Clone the repository:
```bash
git clone https://github.com/wikki2000/ChatWik.git
cd ChatWik
```
## Install dependencies:
```bash
pip install -r requirements.txt
```
## Run the application:
```bash
python3 app.py
```
### API Documentation
API documentation can be accessed while the application is running at: http://127.0.0.1:5001/apidocs/
